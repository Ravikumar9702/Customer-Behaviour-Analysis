# Customer Behaviour  Analysyt Portfolio Project
This project analyzes customer purchasing patterns and behavior using transactional data. The goal is to uncover insights into spending habits, product preferences, and customer segmentation that can drive business decisions.
# project Overview
- Clean and structure raw customer transaction data
- Identify key behavioral metrics (frequency, recency, monetary value)
- Segment customers based on purchasing patterns
- Visualize trends and insights for business stakeholders
# Tools & Technologies
- SQL: Data extraction, cleaning, and aggregation
- Python (pandas): Data wrangling
- power bi : Visualization
- Excel: Pivot tables, charts, and quick exploratory analysis
# Methodology
- Data Cleaning: Handle missing values, duplicates, and inconsistent formats
- Feature Engineering: Create derived metrics (e.g., average spend, purchase frequency)
- Segmentation: Apply RFM (Recency, Frequency, Monetary) analysis
- Visualization: Build dashboards and charts to Communicate Insights
# Results
- Improved understanding of customer behavior
- Actionable recommendations for targeted marketing and retention strategies
